إذا كان الهرم قاعدته دائرية أو بيضوية:
في الإسقاط الثنائي الأبعاد (المثلث):
نرى قطعة مستقيمة (القاعدة = σ = 0.5)

ونرى ارتفاعاً (t)

ونرى وتراً (H)

في الحقيقة ثلاثية الأبعاد (الهرم):
القاعدة ليست قطعة مستقيمة، بل دائرة أو قطع ناقص (Ellipse)

σ = 0.5 قد يكون نصف قطر أو بعداً محورياً

المتجهات تدور على سطح هذا الهرم البيضوي

هذا يفسر عدة أشياء:
1. لماذا σ = 0.5 ثابت؟
في القطع الناقص، المحور شبه الرئيسي (semi-major axis) قد يكون ثابتاً.

2. لماذا تظهر π؟
مساحة الدائرة = πr²، محيط الدائرة = 2πr

3. لماذا "ثابت الهرم" ≈ 1.1؟
1.1 قريب من:

π/3 ≈ 1.047

2/π ≈ 0.636 (لا)

π/√8 ≈ 1.11 ✅

π/√8 = 3.1416 / 2.828 = 1.0472

وهذا قريب جداً من متوسط نسبة موبيوس (~1.1)!

from ZetaLab_Supreme import ZetaLab
from mpmath import mp, zetazero
import numpy as np
import pandas as pd

mp.dps = 50
lab = ZetaLab(precision=50)

results = []

print("Calculating elliptic cone metrics for first 100 zeros...")

for n in range(1, 101):
    rho = zetazero(n)
    t = float(rho.imag)
    z_prime = lab.get_zeta_derivative(rho, order=1)
    abs_z_prime = float(abs(z_prime))
    
    # 1. Base area of elliptic cone: π * t * (1/|ζ'|)
    base_area = np.pi * t * (1.0 / abs_z_prime)
    
    # 2. Ratio t / |ζ'| (aspect ratio of ellipse)
    aspect_ratio = t / abs_z_prime
    
    # 3. Moebius invariance constant (N=10000 fixed)
    N_fixed = 10000
    ln_N = np.log(N_fixed)
    
    # Pre-calculate M_N for this rho (using cached mu if possible)
    mu = lab.sieve_mu(N_fixed)
    n_vals = np.arange(1, N_fixed + 1)
    powers = n_vals ** (-0.5 - 1j * t)
    M_N = np.sum(mu[1:] * powers)
    abs_M_N = float(abs(M_N))
    
    moebius_product = abs_M_N * abs_z_prime
    moebius_ratio = moebius_product / ln_N
    
    # 4. π/√8 constant
    pi_sqrt8 = np.pi / np.sqrt(8.0)  # ≈ 1.0472
    
    results.append({
        "n": n,
        "t": t,
        "|zeta'|": abs_z_prime,
        "Base_Area (π·t/|ζ'|)": base_area,
        "Aspect_Ratio (t/|ζ'|)": aspect_ratio,
        "Moebius_Product": moebius_product,
        "Moebius_Ratio": moebius_ratio,
        "Deviation_from_π/√8": moebius_ratio - pi_sqrt8
    })
    
    if n % 10 == 0:
        print(f"  Processed up to Z{n}")

df = pd.DataFrame(results)
df.to_csv("c:/Users/allmy/Desktop/aaa/sn/tmp/elliptic_cone_analysis.csv", index=False)
